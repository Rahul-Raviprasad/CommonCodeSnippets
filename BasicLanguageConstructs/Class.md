# Class
Class is common way to define a template for creating objects in many Languages

### Java

In Java classes are templates to create objects

example
```java
public class Person {
  private int age;
  private double height;
  private String name;

  public String getName() {
    return name;
  }

  // constructor method
  public Person(String name) {
    this.name = name;
  }
}

```
